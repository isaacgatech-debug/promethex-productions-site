# Promethex Productions Site

Independent website for Promethex Productions, built with React, Vite, and Tailwind CSS.

## Development

```bash
npm ci
npm run dev
```

## Build

```bash
npm run build
```

The production build outputs to `dist/`.

## Deployment

Production site:

```text
https://promethex-productions-site.netlify.app
```

## Repository

This project is an independent copy and is not connected to the original Promethex repository.

Current remote:

```text
https://github.com/isaacgatech-debug/promethex-productions-site.git
```

In this local IDE environment, repository metadata is stored in `.repo-git` because creating a standard `.git` directory is blocked. Use:

```bash
git --git-dir=.repo-git --work-tree=. status
```
